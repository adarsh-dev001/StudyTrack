## See Your Image Above!

This post demonstrates how a featured image can be displayed. The image URL is specified in the `featuredImage` field in the frontmatter of this `.mdx` file.

The `next/image` component in the blog layout handles optimizing and displaying this image.

Remember, for external images to be optimized by `next/image`, their domain needs to be added to the `remotePatterns` in `next.config.ts`.

This particular image is about JEE Main 2024 Prep Tips. You can see how a relevant image enhances the visual appeal of a blog post.
